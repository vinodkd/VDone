VDone

We Done? A todo app built for procrastinators and forgetful people. Flexible scheduling, intrusive reminders, and open-loop tracking — without forcing rigid deadlines.

Features

For procrastinators — it'll let you off the hook, then hold you to it
💤

Snooze

"Just 5 more minutes." Fine. 5, 10, or 15 — pick one. The alarm comes back.

🔀

Defer to a Task

Doing something else right now? Pick it from the alarm screen. The reminder re-surfaces automatically when that task is marked done. Can't escape it.

⏭

Skip This Time

For recurring tasks: skip the current instance without marking it done. The next occurrence is scheduled automatically. No guilt, no broken streak — the task just comes back.

🔗

Condition-Based Scheduling

"I'll do it after X" is now a valid schedule. Chain tasks together — a reminder surfaces only when the preceding task is actually done, with an optional delay (e.g. 1 hour after).

For forgetful people — reminders that actually reach you
⏰

Intrusive Alarms

Full-screen alarm when your phone is locked — wakes the screen, shows over the lock screen. Heads-up notification when the phone is on. You won't miss it.

🔁

Recurring Tasks

Daily, weekly, monthly, yearly — with an optional time of day and days-of-week filter. Set it once; get reminded forever. Resets automatically after each completion.

📱

Home Screen Widget

Today's tasks on your home screen without opening the app. Tap a task to open it. Tap + to add a new one.

💬

Open Loops

Waiting on someone else? Mark the task as an open loop with a follow-up date. The Loops tab keeps everything pending visible so nothing disappears into the void.

Core task management
📋

Nested Tasks

Break any task into subtasks to arbitrary depth. Parent tasks show progress across their children.

📅

Schedule at a Glance

Every task row shows its schedule — "Today at 2:30 PM", "Weekdays at 08:00", "After: morning run +1h". No need to open the task to know when it fires.

📆

Flexible Scheduling

Fixed date/time, recurring with specific days of the week, or condition-based (after task done, before task starts). Pick the right mode per task.

🔍

Search

Filter any task by title or notes from the All Tasks tab. Instant results as you type.

🎤

Show Mode

At a gig or event? Enable Show Mode to silence all alarms globally. Switch it off when you're back.

Installing the APK

VDone is not yet on the Google Play Store. Install it directly using the APK file. This requires enabling installs from unknown sources on your device.

  1. Download the APK

    Tap the "Download APK" button above, or go to the Releases page and download vdone.apk.

  2. Allow installs from unknown sources

    On Android 8+: when prompted after opening the APK, tap Settings and enable "Allow from this source" for your browser or file manager. On older Android: go to Settings → Security → Unknown sources and enable it.

  3. Open the APK and install

    Open the downloaded file from your notifications or the Downloads folder. Tap Install and wait for it to complete.

  4. Grant permissions on first launch

    VDone will ask for Notifications, Exact alarms, and Display over other apps (full-screen intents). All three are needed for reminders to work reliably.
