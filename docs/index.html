VDone

A todo app for procrastinators and forgetful people.
Flexible scheduling, intrusive reminders — no rigid deadlines.

  • Full-screen alarms — wakes the screen, shows over the lock screen; can't be ignored
  • 5-tab workflow — Plan · Start · Doing · Waiting · Done; each an action verb
  • Recurring tasks — daily / weekly / monthly / yearly, specific days, at a set time
  • Snooze, skip, defer — 5/10/15 min snooze; skip a recurrence; defer behind another task
  • Condition-based tasks — fires only after another task is done, with an optional delay
  • Per-task alarm sound — pick a different ringtone so your brain can't tune it out
  • Open loops — waiting on someone? track it with a follow-up reminder
  • Nested subtasks — break tasks down to any depth; parent shows progress
  • Home screen widget — today's tasks at a glance without opening the app
  • Deactivate tasks — pause without deleting; reactivate any time
  • 100% local — no account, no sync, no tracking

Installing the APK

VDone is not yet on the Google Play Store. Install it directly — requires enabling installs from unknown sources.

  1. Download the APK

    Tap "Download APK" above, or get vdone.apk from the Releases page.

  2. Allow installs from unknown sources

    Android 8+: when prompted after opening the APK, tap Settings → enable "Allow from this source". Older Android: Settings → Security → Unknown sources.

  3. Open the APK and install

    Open the downloaded file from your notifications or Downloads folder. Tap Install.

  4. Grant permissions on first launch

    VDone needs Notifications, Exact alarms, and Display over other apps for reminders to work reliably.

About this app

VDone is local only. All your tasks stay on your device. No account required. No data leaves your phone. No analytics, no tracking, no ads.

The code was written by Claude Code. The intent, design, and product decisions were made by a human. Think of it as pair programming where one half of the pair never gets tired.

Source is fully open. From empty Android project to v1.0.34 — no code written by hand.
