VDone

A todo app for procrastinators and forgetful people. Flexible scheduling, intrusive reminders — no rigid deadlines.

⬇ Download APK (Android) How to install ↓ GitHub

😬 For procrastinators

  • Snooze — 5, 10, or 15 min; alarm comes back
  • Defer to a task — pick what you're doing instead; reminder resurfaces when that task is done
  • Skip this time — suppress this instance of a recurring task; next one schedules automatically
  • "I'll do it after X" — chain tasks; reminder fires only after the preceding task is done, with an optional delay

🤔 For forgetful people

  • Full-screen alarm — wakes the screen, shows over the lock screen; can't be ignored
  • Per-task alarm sound — pick a different ringtone per task so your brain can't tune it out
  • Recurring tasks — daily/weekly/monthly/yearly, specific days; resets after each completion
  • Home screen widget — today's tasks without opening the app
  • Open loops — waiting on someone else? set a follow-up date; tracked in the Loops tab
  • Schedule labels on rows — see when each task fires at a glance without opening it

📋 Everything else

  • Nested tasks — subtasks to any depth; parent shows progress
  • Flexible scheduling — fixed date/time, recurring, or condition-based per task
  • Days-of-week filter — recurring tasks on weekdays, weekends, or custom days
  • Search — filter by title or notes across all tasks
  • Show Mode — silence all alarms globally (gigs, meetings, etc.)
  • 100% local — no account, no sync, no tracking

Installing the APK

VDone is not yet on the Google Play Store. Install it directly — requires enabling installs from unknown sources.

  1. Download the APK

    Tap "Download APK" above, or get vdone.apk from the Releases page.

  2. Allow installs from unknown sources

    Android 8+: when prompted after opening the APK, tap Settings → enable "Allow from this source". Older Android: Settings → Security → Unknown sources.

  3. Open the APK and install

    Open the downloaded file from your notifications or Downloads folder. Tap Install.

  4. Grant permissions on first launch

    VDone needs Notifications, Exact alarms, and Display over other apps for reminders to work reliably.

About this app

VDone is local only. All your tasks stay on your device. No account required. No data leaves your phone. No analytics, no tracking, no ads.

The code was written by Claude Code. The intent, design, and product decisions were made by a human. Think of it as pair programming where one half of the pair never gets tired.
