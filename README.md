VDone
====

VDone is a todo application for procastinators. It also helps people do repetitive tasks (eg, daily chores).

Features
-----

* Users can enter todos like any other todolist app, but dont necessarily need to specify a start date/time. 
* Todos could start/end per a schedule, but they dont need to. Instead they could set conditions for when they start, or that they shoudl be done sometime during the day/week/month/year depending on their frequency; or when a task they depend on is complete, or has not yet started.
* Tasks can nest ie have children, indefinitely
* There should be an open loops display for next tasks
* When a task is due, there should be the ability to snooze it and be reminded in x secs/mins/hours/days. 
* Reminders must ring such that they are modal and visible to the user, not just show up as a notification.
* Reminders should be configurable to make sounds, or just vibrate.
